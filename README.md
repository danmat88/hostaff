# HostAff Pro

Hosting affiliate comparison and review app starter built with React + Vite.

## What Is Included

- Production-style landing and navigation shell
- Filterable and sortable hosting comparison table
- Full review card grid with strengths and tradeoffs
- Feature matrix, trust section, FAQ, and newsletter CTA
- Structured local data model for providers, FAQs, reviews, and features

## Project Structure

- `src/data/hostingData.js`: provider catalog and content data
- `src/components/layout`: navigation and footer
- `src/components/sections`: page-level content sections
- `src/components/ui`: reusable UI primitives
- `src/App.jsx`: orchestration, filters, sort logic, and section composition

## Run Locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
```
